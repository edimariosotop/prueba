# Página web de ejemplo

Este repositorio contiene una página web sencilla creada como punto de partida.

## Cómo mantener el proyecto en GitHub

- Realiza commits frecuentes con mensajes claros.
- Usa ramas para nuevas características y _pull requests_ para revisiones.
- Documenta los cambios en este `README`.
- Ejecuta pruebas y validadores antes de subir cambios.
- Mantén tus dependencias actualizadas.

## Visualizar la página

Puedes publicar este sitio con **GitHub Pages** habilitando la opción en la
configuración del repositorio y seleccionando la rama `main`.
